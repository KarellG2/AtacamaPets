# AtacamaPets
Creacion de una Pagina Web para la Clinica Veterinaria Atacama Pets 

Nombre de la Empresa: AtacamaPets

- Descripción del Negocio: AtacamaPets es una clinica veterinaria ubicada en la ciudad de Copiapo, ubicada en la region de Atacama, la cual se encarga en brindar la maxima atencion y bienestar de las mascotas, contando con una gran gama de servivios como consulta general, farmacia y peluqueria canina.

- Necesidad Digital: La empresa carece de tener una presencia web en donde los clientes puedan conocer cada uno de sus servicios, por lo cual se creara un sitio web accesible y que resuelvera la comunicacion faltante.

- Lista de Secciones: Inicio, Servicios, Nosotros, Ubicacion, Contacto, Footer.
